# TrendyMall

Mobile phone accessories storefront — Next.js (App Router) + Tailwind CSS + Supabase.

For everything you need to get this running (Supabase setup, environment
variables, seeding the database, creating your first admin account, adding
products, deploying to Vercel), see **[SETUP.md](./SETUP.md)**.

## Quick start (once Supabase is configured)

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
